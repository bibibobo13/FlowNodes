## 📐 The Polygon Concept
Tasks are represented by their complexity:

*   🔺 **Triangle (3 edges):** Low complexity
*   🔷 **Square (4 edges):** Medium complexity
*   ⬟ **Pentagon (5+ edges):** High complexity

> *"The more sides, the more focus required"*
